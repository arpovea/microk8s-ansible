# Despliegue Tekton

Este rol se encarga del despliegue de Tekton en Microk8s.

## Documentación de variables de Tekton

| Variable        | Descripción                                          | Valor por defecto |
|-----------------|------------------------------------------------------|-------------------|
| `enable_tekton` | Habilita o deshabilita el despliegue de Tekton.     | `false`           |
